# Random-colour-generator
Random color generator with HEX &amp; RGB values. Click button to generate new color, copy code with one click. Built with HTML, CSS &amp; Vanilla JavaScript.
Day 15/100 of #100DaysOfCode. Random color generator that creates HEX and RGB values on button click. Features one-click copy for both formats and live background update. Built to practice Math.random(), color conversions, and Clipboard API. Tech: HTML5, CSS3, Vanilla JavaScript, Math.random(), navigator.clipboard.
